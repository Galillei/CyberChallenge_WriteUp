# No Comments

## Задание

Найдите флаг, скрытый в веб-сайте: <http://nocomments.2018.cyberchallenge.ru/>

## Решение

Название намекает на просмотр комментариев в коде, открываем HTML код сайта и находим флаг: **CC{hide_and_seek_is_sane_enough}**
